    * **개별 공부 진행 후 복습 + 공부 내용 정리 요약 + 관련 프로그래머스 문제풀이**

    * 요약한 사람이 주도로 리뷰(30분)
    * 프로그래머스 풀이 코드리뷰
    * 관련  토이 프로그램 작성
